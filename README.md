# FinanzasApp - Financial Curator

Plataforma de gestión de finanzas personales diseñada para estudiantes y profesionales. Permite llevar un control detallado de ingresos y gastos, organizar categorías y gestionar beneficiarios de forma sencilla y eficiente.

## 🚀 Despliegue
Puedes ver la aplicación en vivo aquí: [FinanzasApp en GitHub Pages](https://Ender-Jesus.github.io/finanzas-app-previo1/)

## ✨ Funcionalidades
- **Autenticación**: Registro e inicio de sesión seguro mediante JWT.
- **Resumen Financiero**: Dashboard con balance total, ingresos y gastos mensuales.
- **Visualización de Datos**: Gráfico interactivo de gastos por categoría utilizando Chart.js.
- **Gestión de Categorías**: Creación y eliminación de categorías personalizadas para ingresos y gastos.
- **Directorio de Beneficiarios**: Administración de contactos frecuentes para registros rápidos.
- **Registro de Movimientos**: Entrada detallada de transacciones con clasificación automática.
- **Filtros Avanzados**: Historial de movimientos filtrable por tipo y rango de fechas.
- **Experiencia de Usuario**: Sistema de notificaciones (toasts), estados de carga y diseño totalmente responsivo.

## 🛠️ Tecnologías
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (CDN).
- **Gráficos**: Chart.js.
- **API**: RESTful API basada en Spring Boot.

## 📂 Estructura del Proyecto
- `index.html`: Punto de entrada con redirección inteligente.
- `pages/`: Contiene todas las vistas de la aplicación.
- `assets/js/`: Lógica de la aplicación, utilidades de API y sesión.
- `assets/img/`: Recursos visuales.
